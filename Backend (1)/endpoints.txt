MAMACARE BACKEND API ENDPOINTS
Generated from current Spring controllers on June 18, 2026

Base URL:
  Development: http://localhost:8080
  API prefix:  /api/v1

Auth rule:
  Public endpoints:
    POST /api/v1/auth/register
    POST /api/v1/auth/verify-otp
    POST /api/v1/auth/authenticate

  Most other endpoints require:
    Authorization: Bearer <access_token>

============================================================

POST /api/v1/auth/register
  Purpose: Start user registration and send OTP.
  Auth: Public
  Controller: AuthenticationController.register

POST /api/v1/auth/verify-otp
  Purpose: Verify OTP and complete registration/login token response.
  Auth: Public
  Controller: AuthenticationController.verifyOtp

POST /api/v1/auth/authenticate
  Purpose: Login with user credentials.
  Auth: Public
  Controller: AuthenticationController.authenticate

POST /api/v1/auth/refresh-token
  Purpose: Refresh access token.
  Auth: Public
  Controller: AuthenticationController.refreshToken
